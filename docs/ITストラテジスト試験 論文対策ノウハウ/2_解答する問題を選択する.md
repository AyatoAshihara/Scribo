# 3.2 解答する問題を選択する
午後II試験は3問の中から1問を選択して論述します。ここでは、問題を選択する基準について説明します。

## 3.2.1 選択する基準を事前に決めておく
午後II試験の解答時間は限られています（2時間）。問題の選択はできるだけ素早く完了させなければなりません。そのためには、事前に選択基準を決めておくのが有効です。

【問題の選択基準】
1. 問題文中の「箇条書き」や「設問イ」を読んで、事前に用意した**「準備モジュール（論述の部品）」**を使って論述できそうだと感じたら候補とする。
2. 問題文中の「箇条書き」や「問題文のテーマ行（1行目）」を読んで、事前に用意した**「背景」**に近いと感じたら候補とする。
3. 自分自身で似た経験をしたことがあれば候補とする。
4. 問題文のテーマ行（1行目）を読んで、類似の過去問題を解いたことがあれば候補とする。

## 3.2.2 選択基準(1)に該当する場合：〈準備モジュールで選ぶ〉
事前に準備した「論文の部品」が、問題文の「箇条書き」や「設問イ」に合致するかを確認します。

事例：令和5年 午後II 問2「個別システム化計画におけるシステムリスク対応方針の立案について」
この問題の「疑似箇条書き」と、準備していた「モジュール」を照合します。

項目,内容（問題文の要素）
疑似箇条書き1,①インシデント発生の最小化、②許容される停止時間に応じた冗長性確保、③権限の厳格化、④迅速な情報共有とリカバリ
疑似箇条書き2,①施策の準備、②実行、③維持、④組織の体制強化などに要する費用の明示
設問イの一部,「個別システムにおいて、あなたはどのようなシステムリスク対応方針を立案したか。…」

【AIへの指示/Few-shotポイント】 事前に準備した「準備モジュール」が、問題文の箇条書き（上記①〜④など）に3つ以上合致する場合、その問題を選択すると論述の成功率が飛躍的に高まります。

## 3.2.3 選択基準(2)に該当する場合：〈背景で選ぶ〉
問題文の1行目（テーマ行）が、事前に準備した「背景（会社概要や現状の課題）」と合致するかを判断します。

背景のコンポーネント例（テンプレート）
論文の導入部（設問ア）でそのまま使える、論理的な背景構成の例です。

[背景の記述例]
- 会社紹介： A社は、大手生命保険会社である。
- 会社の現状： 近年は生命保険商品の多様化が進むと同時に、損害保険などこれまで扱ったことがない他の分野の商品も扱うようになってきている。
- 課題： 営業部門要員の確保が困難になってきており、要員教育コストも重くのしかかっている。
- 解決の方向性： 合理化のため、業務プロセスを変更し、収益性を改善したい。
- 不安材料： 現場からは、要員シフトに対し反発の声が上がっている。

## 3.2.4 選択基準(3)(4)に該当する場合
基準(1)(2)に当てはまらない場合の最終判断基準です。
- 基準(3) 自身の経験： 自分の実務上の体験や、似たような過去問題を解いた経験があれば、設問の要求事項に答えられる可能性が高い。
- 基準(4) 過去問の類似性： 「取っ掛かり」さえあれば、論文の設計書を何とか作成できる見込みが立ちます。

## 3.2.5 演習では何でも対応できるようにする
本番では2分以内に問題を選択することを目標にします。
- 目標： 問題選択は2分以内。
- 本番のルーチン： 受験番号と生年月日の記入、論述対象のシステム概要の記入（約5〜8分）を含め、開始10分以内に構想段階へ入る。

## 統合ガイド：AI用論文生成プロンプト（指示書）
AIに論文を生成させる際、以下の構造を制約条件として与えてください。

【プロンプト構成案】
# 役割
あなたはITストラテジスト試験の合格者です。以下の制約に基づき、合格レベルの論文を生成してください。

# 論文構成のルール
1. 設問ア（800字以内）：
- [背景のテンプレート] を活用し、企業の概要、事業特性、現在直面している課題を具体的に記述すること。
- 「数値（売上、拠点数、従業員数など）」を盛り込み、リアリティを出すこと。

2. 設問イ（800字以上1,600字以内）：
- 問題文の「箇条書き」にある要素（例：冗長性確保、権限厳格化など）を必ず見出しとして取り込むこと。
- 事前に用意した「準備モジュール」の内容を、問題の文脈に沿って具体化すること。

3. 設問ウ（600字以上1,200字以内）：
- 実施した施策に対する「評価」と「今後の課題」を記述すること。
- 経営層や事業部門からのフィードバック（承認、指摘事項）を含めること。

# 禁止事項
- 抽象的な表現（「適切に」「十分に」等）を避け、具体的な行動や基準（「RTOを2時間以内に設定し…」等）を記述すること。
